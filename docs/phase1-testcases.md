# Phase 1 測試清單

## 1. 新增日誌
- [ ] 欄位 1-3 不填會阻止儲存並提示（繁中）
- [ ] 日期預設今天，可修改
- [ ] 機號只能英文+數字（含中文/符號要提示錯誤）
- [ ] 故障碼只能數字（含字母要提示錯誤）
- [ ] 儲存成功後回到列表可看到新資料

## 2. 修改日誌
- [ ] 修改後保存成功，列表與詳情都更新

## 3. 刪除日誌
- [ ] 刪除前有二次確認
- [ ] 刪除後列表消失

## 4. 搜尋（多條件）
- [ ] 任一欄位可被搜尋到
- [ ] 多條件 AND：條件越多結果越少（符合預期）
- [ ] 空結果有提示

## 5. 匯出
- [ ] 可將搜尋結果匯出（Phase 1：複製到剪貼簿或分享）
